쿠키
쿠키 : 웹 브라우저가 보관하는 데이터
웹 브라우저는 웹 서버에 요청을 보낼때 쿠키를 함께 전송
웹 서버에서 웹 브라우저가 전송한 쿠키를 이용해서 필요한 데이터를 읽음
쿠키는 서버에서도 , 브라우저에서도 생성할수 있다.
JSP에서는 서버에서 생성하는 쿠키

쿠키의 구성
이름 : 쿠키를 구별하는데 사용
값 : 해당 쿠키 이름에 설정된 값
유효시간(만료시간) : 쿠키가 유지될 시간
경로 : 쿠키를 전송할 요청 경로
도메인 : 쿠키가 사용될 도메인을 지정

세션
세선 : 클라이언트의 상태를 서버에 저장 (웹 브라우저가 아님)
세션은 웹 브라우저마다 따로 존재, 세션은 서버에만 생성
로그인한 사용자 정보를 세션에 저장 (인증 정보)
세션을 구분하기 위해서 사용하는 ID => Cookie 했을때 봤던 JSESSIONID
JSESSIONID : 쿠키가 세션ID를 공유할때 사용한다.
